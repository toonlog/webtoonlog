## 완성된 기능
- 닉네임+비밀번호 로그인/회원가입
- 웹툰 목록 (장르/플랫폼 필터, 검색)
- 작품 상세 페이지 (장르 태그 클릭 → 장르별 목록)
- 별점/리뷰 (작성/수정/삭제/공개비공개)
- 읽기 상태 (읽는중/완독/읽고싶다/보류)
- 웹툰 등록 (중복 확인 포함)
- 마이페이지 (리뷰/읽기상태 모아보기, 닉네임/프로필사진 변경, 탈퇴)
- 팔로우/언팔로우
- 컬렉션 (생성/수정/삭제, 웹툰 추가/제거, 공개/비공개)
- 유저 프로필 페이지
- 메인 배너 슬라이더 (2초 자동, 모바일 스와이프)
- 이번 주 인기작 섹션
- 모바일 최적화

## 인증
- JWT (app/lib/auth.js) - localStorage에 token, nickname, userId 저장
- API 요청시 Authorization: Bearer {token} 헤더 사용
